# Job Portal Management System

## Overview

The Job Portal Management System is a full-stack web application developed to simplify the recruitment process by connecting job seekers with employers on a single platform. The application provides separate modules for administrators, employers, and job seekers, allowing each user to perform tasks according to their role. It offers an easy-to-use interface for posting jobs, searching for opportunities, applying for jobs, and managing user information.

This project was independently designed and developed as an academic project to strengthen practical knowledge of Java-based web development, database management.

## Features

### Job Seeker Module

* User registration and secure login.
* Create and manage personal profile.
* Browse available job opportunities.
* View detailed job descriptions.
* Apply for jobs online.
* 
### Employer Module

* Employer registration and login.
* Post new job vacancies.
* Edit and update job listings.
* Delete expired or incorrect job postings.
* View applications received from job seekers.
* Manage company information.

## Project Objective

The main objective of this project is to provide an efficient online recruitment platform where employers can publish job opportunities and job seekers can easily discover and apply for jobs. The system reduces manual recruitment efforts, improves communication between recruiters and candidates, and provides a centralized platform for managing the hiring process.
